# Lab 5 – SecDevOps Integration & Continuous Security Monitoring

## Objective

This lab demonstrates:

- CI/CD pipeline setup
- GitHub integration
- GitLab CI configuration
- Security scanning using Snyk
- Continuous security monitoring

---

# Technologies Used

- GitHub
- GitLab CI
- Node.js
- Snyk
- Linux

---

# Steps Performed

1. Created Node.js web application
2. Configured GitLab CI pipeline
3. Installed Node.js and npm
4. Installed Snyk
5. Performed vulnerability scanning
6. Integrated Snyk into CI/CD pipeline

---

# Commands Used

```bash
node index.js
snyk test
npm install
npm test
```

---

# Conclusion

This lab demonstrated how security tools can be integrated into CI/CD pipelines for automated vulnerability detection and continuous monitoring.
